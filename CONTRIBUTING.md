# 🤝 Contributing to VaultIQ

First off, thank you for considering contributing to VaultIQ! It's people like you that make VaultIQ such a great tool for the community.

## 🚀 How Can I Contribute?

### Reporting Bugs
*   Check the [Issues](https://github.com/sahik67/VaultIQ-System/issues) to see if the bug has already been reported.
*   If not, open a new issue with a clear title and detailed description.

### Suggesting Enhancements
*   Open an issue with the tag `enhancement`.
*   Explain why this feature would be useful and how it should work.

### Pull Requests
1.  Fork the repo and create your branch from `main`.
2.  If you've added code that should be tested, add tests.
3.  Ensure the CI/CD health check passes.
4.  Write a clear commit message.

## 📜 Code of Conduct
By participating in this project, you agree to abide by the terms of the MIT License and maintain a respectful environment for all contributors.

## 🛠️ Development Environment
*   **Android**: Android Studio Hedgehog+
*   **Backend**: Supabase CLI
*   **Bot**: Python 3.10+
